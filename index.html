Super Hero Sightings

Super Heroes are all over the place these days. This is an appliation I created to help fans keep track of this abundance of super heroes, their affiliated organizations, and their latest sightings.

I had great fun creating this Spring MVC web application and learned a lot from it.

This project incorporated the following concepts:

  • Creating wireframes to outline the UI and built it with HTML, CSS< and Bootstrap
  • RESTful Application Design; using JavaScript, AJAX, and jQuery
  • Spring MVC architecture; using Java, JSPs and Servlets
  • Unit Testing with the JUnit Framework
  • Tomcat and JDBC to connect to a MySQL database using prepared statements
  • Built the MySQL database with SQL to store a dynamic list of Super Humans and track their sightings, alliances, and abilities
  • Created Custom Error Pages to ensure a better user experience and remove unneccessary information from being leeked to visitors for security
  • Used Spring Security to create a secure login system with different levels of access
  • Used Spring Security BCrypt to secure User's login passwords

Flooring Order Business Application

The goal of this project was to create a CRUD (create, read, update, delete) application that allows for the tracking of a company's flooring orders. This program has two configurations: production and training. It stores data in multiple XML files and is configured so that information can only be saved (written to files) in the production mode.

Concepts covered include:

  • N-Tier/MVC architecture including the use of a Service Layer
  • Interfaces
  • Spring Dependency Injection
  • Spring Aspect-Oriented Programming
  • Unit Testing
  • File I/O with XML files
